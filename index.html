This is my first Website Project!

In this project, I apply what I've learned after a course in Web Design.


Design inspiration

I loved the way that Riot Game's Valorant website looked. So, I tried to mimic their website design with the skills I had, and by inspecting their websites' pages a lot.


Previous website idea

At first, I thought about my website project containing an archive. The website project would be updated every week or to and stored in the archive. This would show what I've learned for Web Design as I progressed throughout the semester. I've kept the previous website project in the Scrapped Website Project page.

Current website

I've scrapped the idea since it took too much effort to create different versions of the website project to store in the archives, and to edit each of them. I also wanted to remake my website from the ground up in order to clean up my code a bit and to add in more CSS. The result is this website project.
