The Purpose of This Website

This website was created for a Web Design course's final project. In this project, I a bit of what I've learned throughout the semester , but I have not included everything in it.


What I Envisioned for the Website

When figuring out how I wanted my website to look like, I took inspiration from Valorant's home page. I wanted the website to have a clean, modern, and minimalistic style to it using a light-ish red, white, and black color.


What I've Learned

Through a lot of time spend trying to mimic Valorant's, I learned that Web Design can be difficult. As someone that had just learned about Web Design, it's one thing to have image of the website in your head, and it's another to actually create that using HTML and CSS.

  • creating websites a bit quicker.
  • learning to control the positions of elements better.
